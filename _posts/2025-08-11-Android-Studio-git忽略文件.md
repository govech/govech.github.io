### 根目录忽略文件

```
# ==============================================================================
# Android 项目根目录 .gitignore
# 作用范围：整个项目的所有模块
# ==============================================================================

# ------------------------------------------------------------------------------
# 构建系统相关
# ------------------------------------------------------------------------------
.gradle/
build/
/*/build/

# Gradle Wrapper (保留 jar 文件，忽略下载的分发包)
!gradle/wrapper/gradle-wrapper.jar

# ------------------------------------------------------------------------------
# IDE 相关
# ------------------------------------------------------------------------------
# Android Studio
*.iml
*.iws
*.ipr
.idea/
# 保留代码风格配置（团队共享）
!.idea/codeStyles/
# 保留检查配置（团队共享）
!.idea/inspectionProfiles/


# ------------------------------------------------------------------------------
# 本地配置文件
# ------------------------------------------------------------------------------
local.properties
#gradle.properties

# ------------------------------------------------------------------------------
# 系统文件
# ------------------------------------------------------------------------------
.DS_Store
Thumbs.db
desktop.ini

# ------------------------------------------------------------------------------
# 日志和临时文件
# ------------------------------------------------------------------------------
*.log
*.tmp
*.temp
*~



# ------------------------------------------------------------------------------
# 其他工具
# ------------------------------------------------------------------------------
# Fastlane (如果使用)
fastlane/report.xml
fastlane/Preview.html
fastlane/screenshots
fastlane/test_output

# Fabric/Crashlytics (如果使用)
fabric/
CrashlyticsBuild.properties

# SonarQube (如果使用)
.sonar/

# ------------------------------------------------------------------------------
# 敏感文件 (通常应该忽略)
# ------------------------------------------------------------------------------
# Google Services 配置
google-services.json

# Keystore 文件
*.jks
*.keystore

# 环境配置文件
.env
.env.local
.env.*.local
```

```
# ==============================================================================
# Android 模块级 .gitignore (app/)

# ==============================================================================

# ------------------------------------------------------------------------------

/build

# ------------------------------------------------------------------------------
# 模块特定输出目录
# ------------------------------------------------------------------------------
/outputs/

/debug/

# 生成的文件
# ------------------------------------------------------------------------------
/generated/
/build/generated/

# ------------------------------------------------------------------------------
# 模块特定的敏感文件
# ------------------------------------------------------------------------------
# 模块特定的配置文件 (如果存在)
module-config.properties

# 模块特定的签名配置 (如果存在)
signing-config.json

# ------------------------------------------------------------------------------
# 测试相关
# ------------------------------------------------------------------------------
/reports/
/test-results/

# 临时文件
# ------------------------------------------------------------------------------
*.log
*.tmp
```
